# Churn-rate-Analysis-power-bi-project
## Telecom Customer Churn Analysis Dashboard

## Brief Summary
An interactive Power BI dashboard project analyzing telecom customer churn, customer risk levels, and service usage behavior.

## Overview
This project focuses on identifying customer churn patterns and analyzing factors affecting customer retention in a telecom company using Power BI visualizations and KPIs.

## Problem Statement
Customer churn is one of the biggest challenges for telecom companies. The objective of this project is to identify risky customers, analyze churn behavior, and discover important business insights that can help improve customer retention.

## Dataset
The dataset contains:
- Customer demographic details
- Contract information
- Payment methods
- Internet and phone services
- Tenure details
- Churn records
- Risk scores
- Customer charges

## Tools and Technologies
- Power BI
- Excel / CSV
- Data Visualization
- DAX

## Dashboard Features
- Customer profile dashboard
- Churn probability tracking
- Risk analysis dashboard
- Contract type analysis
- Gender-wise churn analysis
- Service usage insights
- Payment method analysis
- Tenure vs churn trends
- Interactive filters and slicers

## Key Insights
- Total customer records: 7043
- Total churned customers: 1869
- Customers with shorter tenure showed higher churn rates
- Month-to-month contracts had the highest churn probability
- Most churned customers preferred electronic check payment method
- Customers without online security and tech support had higher churn rates
- Fiber optic users showed higher churn behavior compared to other internet services
- Around 2900 customers were categorized as risky customers

## Output
Interactive Power BI dashboards providing customer segmentation, churn prediction insights, and business performance analysis.

## How to Run this Project
1. Download the `.pbix` file
2. Open the file in Power BI Desktop
3. Refresh the dataset if required
4. Interact with filters and visuals

## Results & Conclusion
The analysis highlights the major factors contributing to customer churn. Contract type, payment method, tenure, and service usage significantly influence customer retention. Businesses can use these insights to reduce churn and improve customer satisfaction.
